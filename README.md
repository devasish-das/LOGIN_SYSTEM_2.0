# Login System

A simple PHP and MySQL based login and registration system with session authentication.

## Features

- User Registration
- User Login
- Session Management
- Logout Functionality
- Password Validation
- Protected Dashboard Page
- MySQL Database Integration
- Error and Success Messages

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- XAMPP
